# iOS 快捷指令配置指南 — 美团骑车锁车提醒

> 适用版本：iOS 17+（iOS 16 也支持，但部分界面略有差异）
>
> ⏱ 预计配置时间：5 分钟

---

## 配置概览

本快捷指令会在你**打开美团 App** 时自动触发，让你选择预计骑行时长，然后在系统提醒事项中创建一个 X 分钟后的提醒。

## 前置条件

- [ ] iPhone 已安装 **美团 App**
- [ ] iPhone 系统版本 **iOS 17+**
- [ ] 快捷指令 App（系统自带，无需安装）

---

## 分步配置

### 第一步：创建自动化

1. 打开 **快捷指令（Shortcuts）** App
2. 点击底部 **自动化（Automation）** 标签
3. 点击右上角 **+** → **创建个人自动化（Create Personal Automation）**

### 第二步：设置触发器

1. 在列表中找到并选择 **App**
2. 点击 **选取（Choose）**，在搜索框中输入"美团"
3. 勾选 **美团**（图标为黄色美团外卖/单车 Logo）
4. 勾选 **已打开（Is Opened）**
5. 🔘 **取消勾选"立即运行（Run Immediately）"**— 等等，这里需要注意：
   - 如果你希望无感自动运行，保持 **立即运行** 勾选
   - 如果不勾选，每次打开美团都会弹窗确认，比较烦
   - **建议保持勾选"立即运行"**
6. 点击 **下一步（Next）**

> 💡 **提示**：iOS 17+ 的"立即运行"模式下自动化也会弹出通知横幅，但无需你点击确认即可执行。

### 第三步：添加动作

点击 **添加操作（Add Action）**，按顺序添加以下动作：

#### 动作 1：获取当前日期

- 搜索 **"当前日期"** → 选择 **获取当前日期（Get Current Date）**
- 日期格式保持默认即可

#### 动作 2：弹出菜单选择时长

- 搜索 **"列表"** → 选择 **从列表中选择（Choose from List）**
- 在下方输入框中，每行一个选项：
  ```
  15 分钟
  20 分钟
  30 分钟
  ```
- 取消勾选 **选择多项目（Select Multiple）**

#### 动作 3：计算提醒时间

- 搜索 **"计算"** → 选择 **计算（Calculate）**
- 点击第一个变量输入框 → 选择 **当前日期（Current Date）**
- 点击运算符号 → 选择 **+（加号）**
- 点击第二个变量输入框 → 选择 **选择的列表项（Chosen Item）**
- 这样就会把当前时间 + 选择的时长得到提醒时间

#### 动作 4：拆分日期获取提醒时间

- 搜索 **"拆分日期"** → 选择 **拆分日期（Get Date Components）**
- 点击 **当前日期** 变量 → 选择上一步的 **计算结果（Calculation Result）**
- 格式选择 **自定义（Custom）**
- 在自定义格式中输入：`HH:mm`（这将返回类似"14:30"的时间字符串）

#### 动作 5：创建提醒事项

- 搜索 **"提醒事项"** → 选择 **添加提醒事项（Add Reminder）**
- 设置以下字段：

| 字段 | 值 |
|------|-----|
| **标题** | `🚲 美团骑车 — 该锁车了！` |
| **备注** | `你在 %Chosen Item% 前的骑行，请记得锁车！` |
| **提醒时间** | 选择 **日期** → 选择 **计算后的结果（Calculation Result）**（即动作 3 的结果） |
| **列表** | 选择已创建的"美团提醒"列表（建议提前在提醒事项 App 中新建一个"美团提醒"列表） |
| **优先级** | 高 |

#### 动作 6：显示确认通知

- 搜索 **"显示通知"** → 选择 **显示通知（Show Notification）**
- 标题：`✅ 锁车提醒已设置`
- 正文：`将在 X 分钟后提醒您锁车`（点击正文 → 选择变量 → **选择的列表项（Chosen Item）**）

### 第四步：完成创建

1. 点击 **下一步（Next）**
2. 🔘 关闭 **运行前询问（Ask Before Running）**（非常重要！否则每次都要手动确认）
3. 点击 **完成（Done）**

## 使用效果

当你打开美团 App 时：

1. **无感运行**：自动化在后台静默执行
2. **菜单弹出**：屏幕顶部弹出选择器，让你选 15/20/30 分钟
3. **确认通知**：选择后立即弹出 ✅ 通知确认已设置
4. **到点提醒**：到时间后，系统提醒事项弹出通知（锁屏状态下也会显示）
5. **锁车后**：打开提醒事项 App，手动标记提醒为"完成"即可停止

## 常见问题

### Q：每次点外卖也触发，怎么办？

这是已知的**设计取舍**。打开美团 App（无论什么目的）都会触发，相比忘记锁车造成持续计费，偶尔的点外卖误触发是可以接受的。

如果你确实想区分，可以考虑以下变通方案：
- 在动作 2 中增加一个"只是点外卖"的选项，选择后创建一个 1 分钟后的提醒然后快速取消
- 或者在弹出菜单时直接点右上角的 **取消（Cancel）** 即可不设置提醒

### Q：如何在锁车后取消提醒？

**方法一（推荐）**：在自动化开始处添加一个条件判断，检查是否有未完成的提醒，如果有则不重复创建。

**方法二**：
1. 打开 **提醒事项（Reminders）** App
2. 找到"🚲 美团骑车"提醒
3. 左滑或点击圆圈标记为"完成"

### Q：消息没有弹出来？

- 检查 iOS **设置 → 通知 → 提醒事项** → 确保通知已开启
- 检查 **设置 → 专注模式** → 是否将提醒事项加入了允许列表
- 确保在配置自动化时关闭了"运行前询问"

### Q：我选了时长之后，快捷指令卡住了？

iOS 快捷指令在弹出菜单时有时需要等待几秒钟。如果长时间无响应，可以尝试：
- 锁屏再解锁后重试
- 在快捷指令 App 中手动运行一次测试（点击自动化 → 立即运行）

## 进阶技巧

### 创建专用提醒列表

1. 打开 **提醒事项** App
2. 点击左下角 **添加列表（Add List）**
3. 名称：`美团提醒`
4. 选择喜欢的颜色和图标
5. 在创建快捷指令时，动作 5 的"列表"字段选择这个新列表

这样所有锁车提醒都会归在一个列表里，方便统一管理。

### 设置多个常用时长

你可以自定义菜单选项，支持任意分钟数，例如：
```
10 分钟（短途）
20 分钟（默认）
30 分钟（长途）
自己输入
```

最后一项"自己输入"可以让用户自由输入任意时长。
