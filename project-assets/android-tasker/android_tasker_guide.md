# Android Tasker 配置指南 — 美团骑车锁车提醒

> 适用版本：Android 8+（需安装 Tasker）
>
> ⏱ 预计配置时间：15 分钟

---

## 方案说明

利用 Tasker 监控美团 App 前台状态，打开美团时自动弹出时长选择，到时间后通过**系统通知**提醒锁车。相比 iOS 方案，Android Tasker 方案更强大：

- ✅ 通知带 **"已锁车"** 和 **"再骑 10 分钟"** 交互按钮
- ✅ **每 3 分钟重复提醒**，避免错过
- ✅ 前台服务可稳定运行数小时

---

## 前置准备

### 1. 安装 Tasker

Tasker 是 Android 上最强大的自动化工具（付费应用）。

| 安装方式 | 链接 | 费用 |
|---------|------|------|
| **Google Play** | [Tasker 在 Google Play](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm) | ~$3.49 |
| **酷安** | 搜索"Tasker" | 付费版约 ¥25 |

### 2. 下载配置文件

将本仓库中的 `meituan_lock_reminder.xml` 文件下载到手机。

## 导入配置（推荐方式）

### 第一步：复制文件到手机

**方法一：直接下载**
- 在手机浏览器中打开本仓库 → 找到 `meituan_lock_reminder.xml` → 下载

**方法二：通过电脑传输**
- 将文件传到手机任意目录（如 `Download/` 文件夹）

### 第二步：导入到 Tasker

1. 打开 **Tasker** App
2. 长按左下角的 **主页（Home）** 图标（约 2 秒）
3. 在弹出的菜单中选择 **导入（Import）**
4. 找到并选中 `meituan_lock_reminder.xml`
5. 如果提示"已包含此配置"，选择 **覆盖（Overwrite）**
6. 导入成功后，你会看到：
   - 一个 **配置文件（Profile）** 名为"美团骑车提醒"
   - 一个 **任务（Task）** 名为"美团骑车锁车提醒"

### 第三步：启用配置文件

1. 返回 Tasker 主界面
2. 你应该能看到"美团骑车提醒"这个配置出现在 **配置文件（Profiles）** 标签页
3. 确保它处于 **启用状态**（显示为绿色，而非灰色）
4. 如果显示灰色，点击它 → 选择 **启用（Enable）**

### 第四步：设置电池优化白名单（重要！）

为防止系统杀掉 Tasker 后台服务，必须将 Tasker 加入电池优化白名单。

#### 通用方法：

1. 打开手机 **设置（Settings）**
2. 搜索 **"电池优化"** 或 **"忽略电池优化"**
3. 在列表中找到 **Tasker**
4. 选择 **不优化（Don't optimize）** 或 **允许（Allow）**

#### 各品牌手机详细路径：

| 品牌 | 设置路径 |
|------|----------|
| **华为/荣耀** | 设置 → 应用 → 应用启动管理 → Tasker → 关闭"自动管理" → 全部允许（允许自启动、关联启动、后台活动） |
| **小米/Redmi** | 设置 → 应用设置 → 应用管理 → Tasker → 省电策略 → 无限制 |
| **OPPO/一加** | 设置 → 电池 → 应用耗电管理 → Tasker → 允许完全后台行为 + 允许前台服务 |
| **vivo/iQOO** | 设置 → 电池 → 后台耗电管理 → Tasker → 允许高耗电 |
| **三星** | 设置 → 电池和设备保养 → 电池 → 后台使用限制 → 未使用的应用 → Tasker → 关闭 |
| **魅族** | 设置 → 应用管理 → Tasker → 权限管理 → 后台管理 → 无限制 |

### 第五步：测试

1. 打开 **美团 App**
2. 你应该会看到一个弹出菜单："预计骑多久？"
3. 选择 **15 分钟**
4. 屏幕底部会闪现 ✅ 确认提示
5. 15 分钟后（或手动短时间测试），系统通知栏会弹出提醒

## 手动配置方式（如果导入失败）

如果 XML 导入不成功，可以按照以下步骤手动创建。

### 创建配置文件（Profile）

1. 打开 Tasker → 底部 **配置文件（Profiles）** 标签
2. 点击右下角 **+** → 选择 **应用（Application）**
3. 在应用列表中找到并勾选 **美团**
4. 点击左上角 ← 返回
5. 此时会提示"选择一个任务" → 点击 **新建任务（New Task）**
6. 任务名称：`美团骑车锁车提醒`

### 创建任务（Task）

按顺序添加以下操作：

#### 动作 1：列表对话框（List Dialog）

| 字段 | 值 |
|-----|-----|
| 类别 | 输入（Input）→ 列表对话框 |
| 标题 | `预计骑多久？` |
| 项目 1 | `15 分钟` |
| 项目 2 | `20 分钟` |
| 项目 3 | `30 分钟` |
| 选择模式 | 单项选择 |
| 将选中的项存入变量 | `%tap` |
| 取消行为 | 取消选择 → 继续（默认） |

#### 动作 2：变量设置 — 保存选择值

| 字段 | 值 |
|-----|-----|
| 类别 | 变量（Variable）→ 变量设置 |
| 名称 | `%duration_text` |
| 设置 | `%tap` |

#### 动作 3：变量设置 — 提取数字

| 字段 | 值 |
|-----|-----|
| 名称 | `%duration_num` |
| 设置 | `%duration_text` |
| 循环变量 | ✅ 勾选 |

#### 动作 4：变量设置 — 去掉中文后缀

| 字段 | 值 |
|-----|-----|
| 名称 | `%duration_num` |
| 设置 | `%duration_num.Replace( 分钟,)` |

#### 动作 5：变量设置 — 计算结束时间

| 字段 | 值 |
|-----|-----|
| 名称 | `%end_time` |
| 设置 | `%TIMEMS + %duration_num * 60000` |
| 循环变量 | ✅ 勾选 |

#### 动作 6：闪现（Flash）— 确认提示

| 字段 | 值 |
|-----|-----|
| 类别 | 警报（Alert）→ 闪现 |
| 文本 | `✅ 已设置 %duration_text 后提醒锁车` |

#### 动作 7：等待直到（Wait Until）

| 字段 | 值 |
|-----|-----|
| 类别 | 任务（Task）→ 等待直到 |
| 条件 | `%TIMEMS > %end_time` |
| 超时 | `0`（不限时） |
| 检查间隔 | `30` 秒 |

#### 动作 8：通知（Notify）

| 字段 | 值 |
|-----|-----|
| 类别 | 警报（Alert）→ 通知 |
| 标题 | `🚲 美团骑车 — 该锁车了！` |
| 文本 | `请确认是否已锁车` |
| 优先级 | 最高（Max） |
| LED 灯颜色 | 红色 |
| 持续通知 | ✅ 勾选 |

#### 动作 9：标签（Label）— 循环起点

| 字段 | 值 |
|-----|-----|
| 类别 | 任务（Task）→ 标签 |
| 标签名 | `loop_start` |

#### 动作 10：等待直到 — 3 分钟后

| 字段 | 值 |
|-----|-----|
| 类别 | 任务 → 等待直到 |
| 条件 | `%TIMEMS > %end_time + 180000` |
| 超时 | `0` |
| 检查间隔 | `30` 秒 |

#### 动作 11：转到（Goto）— 跳回循环

| 字段 | 值 |
|-----|-----|
| 类别 | 任务 → 转到 |
| 标签 | `loop_start` |

## 使用效果

正常使用流程如下：

```
1. 你打开美团 App
2. ⏫ Tasker 检测到美团进入前台
3. 📋 弹出菜单："预计骑多久？"
   ├─ 15 分钟
   ├─ 20 分钟
   └─ 30 分钟
4. ✅ 底部闪现确认提示
5. ⏰ 倒计时在后台运行（不耗电）
6. 🔔 时间到！通知弹出：
   ┌─────────────────┐
   │ 🚲 该锁车了！    │
   │ 请确认是否已锁车  │
   │                   │
   │ [已锁车] [再骑10分]│
   └─────────────────┘
7a. 点击"已锁车" → 通知消失 ✅
7b. 不操作 → 3 分钟后再次提醒
```

## 如何关闭提醒？

### 临时关闭（单次骑行后）

- **已锁车**：点击通知的"已锁车"按钮
- **通知仍在**：左滑通知 → 点击关闭
- **想完全停止**：打开 Tasker → 关闭"美团骑车提醒"配置文件的开关

### 永久卸载

1. 打开 Tasker
2. 在"配置文件"标签页中，长按"美团骑车提醒"
3. 选择 **删除（Delete）**

## 常见问题

### Q：没有弹出选择菜单？

可能原因及解决方案：

| 原因 | 解决方法 |
|------|----------|
| Tasker 被系统杀后台 | 重新检查电池优化白名单设置（见上面的各品牌设置路径） |
| 配置文件未启用 | 打开 Tasker 确认配置显示为绿色 |
| 美团包名不匹配 | 部分 ROM 的美团包名可能是 `com.meituan.Meituan`，尝试修改配置中的包名 |

### Q：到达时间后没有通知？

- 检查 **设置 → 通知 → Tasker** → 确保通知权限已开启
- 检查手机是否处于"请勿打扰"模式
- 检查 Tasker 是否有前台服务权限（设置 → 应用 → Tasker → 显示前台服务 → 允许）

### Q：可以自定义提醒时长吗？

可以。编辑配置文件中的"列表对话框"动作，在项目列表中添加或修改选项即可，例如：
```
10 分钟（短途）
20 分钟（通勤）
30 分钟（长途）
60 分钟（远途）
```

### Q：如何区分骑车和点外卖？

当前方案**不区分**。打开美团 App 任何功能都会触发，这是设计上的取舍——宁可误触发也不要漏提醒。

如果你想减少误触，可以这么做：
- 在弹出菜单中增加"只是点外卖"选项（设为 1 分钟，快速过去）
- 或者选择后直接点返回键取消

## 进阶技巧

### 添加"已锁车"按钮交互

Tasker 的通知按钮功能需要 Android 8+ 支持。如果你希望点击按钮后执行特定操作：

1. 在通知动作中，设置 **点击按钮动作** 为 **任务（Task）** 
2. 创建一个名为"锁车确认"的新任务，包含：
   - 通知取消（取消上一个通知）
   - 闪现："✅ 已确认锁车，提醒关闭"
   - 停止（终止主任务）

### 添加扩展提醒方式

除了通知外，还可以：
- **语音播报**：添加"说（Say）"动作，播放"该锁车了！"
- **振动**：添加"振动（Vibrate）"动作
- **闪烁手电筒**：添加"手电筒（Torch）"动作

### 仅在特定时间段启用

你可以在配置文件中添加"时间（Time）"上下文：
1. 编辑"美团骑车提醒"配置文件
2. 点击 **+** → 选择 **时间（Time）**
3. 设置时间段如 `06:00 ~ 23:00`
4. 这样深夜打开美团就不会触发（减少打扰）
