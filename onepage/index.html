01

全部功能

01
⏱️
一键启动倒计时
点击图标自动启动 50 分钟系统倒计时。无弹窗打扰，支持灵动岛实时显示。
02
📸
自动打开扫一扫
倒计时启动后自动跳转美团扫一扫界面。不需任何额外操作，扫码即走。
03
🔔
准时响铃提醒
倒计时归零时系统自动响铃，提醒锁车。再也不会因忘记而多扣费用。
04
🧠
灵动岛支持
倒计时在灵动岛中实时展示剩余时间。无需打开 App，一眼可见进度。
05
🛡️
首次授权引导
首次使用自动引导通知权限授权。简洁明了，一次配置永久使用。
06
⚡
轻量无后台
App 仅 2MB，启动即走。不驻留后台、不耗电、不占内存。用完即走。
02

快速上手

01
📲
安装 App
从 Releases 下载 APK，在手机上完成安装。支持 Android 7.0+ / HyperOS 3。
02
🔑
授权通知
首次打开 App → 点击按钮 → 按系统提示授予通知权限。只需一次。
03
🚲
扫码骑车
点击 App 图标 → 自动启动倒计时 → 自动跳转美团扫一扫 → 扫码骑车出发。
04
🔔
到点锁车
50 分钟到 → 系统响铃提醒 → 拿出手机锁车 → 骑行完美结束。
安装 APK → 授权通知 → 点击图标 → 倒计时启动 → 扫码骑车 → 响铃锁车
03

安装指南

  • 1
    下载 APK
    从 GitHub Releases 下载最新版本 meituan-bike-reminder-v2.7.1.apk
  • 2
    开启安装权限
    在手机设置中允许「未知来源应用安装」—— 或直接使用系统文件管理器安装
  • 3
    打开 App
    点击图标 → 第一次使用需要授予通知权限 → 按提示操作即可
  • 4
    开始使用
    以后每次骑车：点击图标 → 自动倒计时 → 自动跳转美团扫一扫 → 出发！
